Configuración de Azure Integration

Paso 1: Ve a portal.azure.com y entra en Entra ID (Active Directory) > App Registrations.
Paso 2: Crea un "New Registration". Ponle nombre "Outlook Flow" y elige "Accounts in any organizational directory (Any Microsoft Entra ID tenant - Multitenant)".
Paso 3: En "Redirect URI", elige SPA (Single Page Application) y pon la URL de esta app (normalmente http://localhost:5173 o similar).

"Hola IT. Necesito registrar una aplicación SPA (Outlook Flow) en Azure para sincronizar mi calendario con una herramienta de productividad local. Los permisos necesarios son: User.Read y Calendars.ReadWrite. El Redirect URI será: [Poner la URL de esta app]"
